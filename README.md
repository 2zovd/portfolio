# dmytrotuzov.dev

Personal professional website — portfolio, blog, and landing page.

## Stack

Astro 6 · Vue 3 · TypeScript · SCSS · Vitest · Cloudflare Pages

## Development

Built with [Claude Code](https://claude.ai/code) and [GitHub Copilot](https://github.com/features/copilot).

```bash
pnpm install
pnpm dev
pnpm test:run
pnpm build
```

## Structure

Feature Sliced Design principles adapted for Astro. See `agent_docs/architecture.md` for layer rules.
